Título de la app: Analizador Financiero

Objetivo:
Desarrollar una aplicación inteligente capaz de analizar, clasificar y visualizar movimientos de gastos personales y empresariales provenientes de diversas fuentes (billeteras virtuales, resúmenes de tarjetas, home banking, etc.), brindando recomendaciones personalizadas para optimizar el ahorro y la gestión financiera.

Características principales:
- Clasificación automática de gastos mediante IA, interpretando el contexto de cada movimiento y asignando categorías relevantes.
- Integración inicial con MercadoPago (API y MCP), con posibilidad de sumar otras billeteras y bancos en futuras versiones.
- Importación de gastos desde archivos PDF, imágenes y otros formatos, utilizando reconocimiento óptico de caracteres (OCR) y procesamiento inteligente para extraer información relevante (monto, fecha, comercio, categoría, etc.).
- Chat integrado con IA (Azure Open API) para consultas avanzadas sobre gastos, tendencias, patrones y recomendaciones. Ejemplo de preguntas: "¿Cuándo gasté más en supermercados?", "¿Cuál fue mi mayor gasto en septiembre?", "¿Qué gastos puedo reducir este mes?".
- ABM (Alta, Baja, Modificación) manual de gastos para complementar la importación automática y permitir ajustes personalizados.
- Módulo de reportes con gráficos interactivos, métricas clave (total gastado, promedio mensual, gastos por categoría, evolución temporal, etc.) y exportación de informes en formatos PDF/Excel.
- Sistema de alertas y notificaciones inteligentes para advertir sobre gastos inusuales, límites de presupuesto, vencimientos de pagos y oportunidades de ahorro.
- Gestión de usuarios y perfiles, permitiendo la personalización de objetivos financieros, categorías de gastos y preferencias de análisis.
- Seguridad y privacidad: cifrado de datos sensibles, autenticación segura y cumplimiento de normativas vigentes.
- Arquitectura escalable y modular, preparada para integrar nuevas fuentes de datos y funcionalidades en futuras iteraciones.
- Ciclo de vida de desarrollo incremental ágil, utilizando Epics, Features y User Stories para el relevamiento y seguimiento de requisitos.

Funcionalidades clave adicionales:
- Dashboard principal con resumen financiero y acceso rápido a las funciones más utilizadas.
- Filtros avanzados para búsquedas y análisis por fechas, categorías, comercios, montos, etc.
- Histórico de gastos y tendencias, con posibilidad de comparar períodos y detectar cambios significativos.
- Recomendaciones personalizadas basadas en hábitos de consumo y objetivos definidos por el usuario.
- Lenguaje de la aplicación: Español

Tecnologías y herramientas:
- Integración con Azure Open API para procesamiento de lenguaje natural y generación de recomendaciones inteligentes.
- Utilización de frameworks modernos para frontend y backend, asegurando una experiencia de usuario fluida y segura.
- Implementación de pruebas automatizadas en todas las fases del ciclo de vida.

Modelo de ciclo de vida:
- Desarrollo incremental ágil, con entregas frecuentes y validación continua de funcionalidades.
- Relevamiento de requisitos mediante Epics, Features y User Stories, asegurando trazabilidad y alineación con las necesidades del usuario final.
